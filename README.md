# Aplikasi Web Tuntunan Tata Cara Sholat Sesuai Sunnah Rasulullah ﷺ

## Deskripsi

Aplikasi web yang dirancang untuk membantu pengguna mempelajari tata cara sholat sesuai sunnah Rasulullah ﷺ. Informasi disajikan dalam bentuk gerakan, bacaan Arab, transliterasi Latin, dan terjemahan.

## Fitur Utama

* Daftar gerakan sholat
* Detail gerakan sholat
* Bacaan Arab
* Transliterasi Latin
* Terjemahan
* Mode Dewasa
* Mode Anak-anak

## Teknologi

* Laravel
* PHP
* MySQL
* Tailwind CSS
