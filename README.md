# Aplikasi Web Tuntunan Tata Cara Sholat Sesuai Sunnah Rasulullah ﷺ

## Deskripsi

Aplikasi web yang dikembangkan untuk membantu pengguna mempelajari tata cara sholat sesuai sunnah Rasulullah ﷺ. Aplikasi menyajikan urutan gerakan sholat, bacaan Arab, transliterasi Latin, dan terjemahan dalam bahasa Indonesia.

## Tujuan

* Membantu pengguna memahami tata cara sholat yang benar.
* Menyediakan informasi gerakan dan bacaan sholat secara terstruktur.
* Menjadi media pembelajaran yang mudah diakses melalui web.

## Teknologi yang Digunakan

* Laravel 12
* PHP
* MySQL
* Tailwind CSS
* Git & GitHub

## Fitur Utama

* Halaman Beranda
* Daftar Gerakan Sholat
* Detail Gerakan Sholat
* Bacaan Arab
* Transliterasi Latin
* Terjemahan Bahasa Indonesia
